## How to Download & Save

### Quick Method 1: Copy-Paste
1. Open article in browser
2. Select all (Cmd+A)
3. Copy (Cmd+C)
4. Create `.md` file in appropriate tier folder
5. Paste (Cmd+V)
6. Add header with source info

### Quick Method 2: GitHub README Direct
For GitHub repos, navigate to README.md and click the "Copy raw file" button in the top right of the code view.

### Method 3: Use curl (for tech-savvy)
```bash
# Example: Download Agentii-Kit README
curl -s "https://raw.githubusercontent.com/agentii/agentii-kit/main/README.md" > "/Users/frank/X/agentii-kit/specs/003-single-blog/research/external-articles/tier-1-official/agentii-kit-readme.md"
```
