# Newsletter Template (Weekly Digest)

Use this structure for the Substack-style weekly digest.

## 1. Opening Hook (~50 words)

- Timely update or observation.
- Connect to spec-driven work or kits.

## 2. Featured Post (~100 words)

- Title and 1â€“2 sentence summary.
- One key takeaway.
- Link: "Read the full post".

## 3. Kit Spotlight (~50 words)

- Short description of one kit (PMF-Kit, Blog-Tech-Kit, Twitter-Init-Kit, etc.).
- What problem it solves.
- Link to repo or docs.

## 4. Community Shoutout (~30 words)

- Highlight a contributor, user story, or community win.

## 5. CTA (~20 words)

- Ask for replies, feedback, or shares.
- Optionally invite readers to star the repo or try a kit.

Reuse this structure each week for consistency.
