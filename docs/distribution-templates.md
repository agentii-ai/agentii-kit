# Distribution Templates

Use these templates for consistent distribution across channels when a new post goes live.

## 1. Twitter/X Thread (Day 0)

Main tweet:

- Hook: one sharp sentence about the problem.
- One-line benefit.
- Link to the post.

Follow-up tweets (5–7):

1. Key insight or stat.
2. Short explanation of the spec-driven/kits angle.
3. Concrete example from the post.
4. Screenshot or diagram (optional).
5. CTA: "Read the full walkthrough".

## 2. LinkedIn Post (Day 0–1)

- 1–2 sentence hook.
- 3–5 bullets with key ideas.
- Link to the post.
- Soft CTA ("Curious how this works in your team?").

## 3. Dev.to Cross-Post (Day 1–2)

- Use the original markdown body.
- Add frontmatter with `canonical_url` pointing back to the main blog.
- At the end, add a short note:
  - "This article originally appeared on kits.agentii.ai" with a link.

## 4. Slack/Discord Share (Day 1–2)

- One-sentence context tailored to the community.
- The link.
- Optional: one bullet with what readers will learn.

## 5. Newsletter Digest Snippet

- 1–2 sentence summary of the post.
- One key quote or stat.
- Link with a clear CTA ("Read the full guide").

Use these templates as starting points and adjust wording for each specific post and audience.
